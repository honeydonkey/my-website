Network Traffic Analysis Tool

Python Machine Learning Network Security Wireshark Data Analysis

Project Overview

The Network Traffic Analysis Tool is a Python-based application designed to analyze network traffic patterns and identify potential security threats using machine learning algorithms. This tool helps security analysts detect anomalies, potential intrusions, and suspicious activities in network traffic data.

Problem Statement

Traditional network monitoring tools often rely on signature-based detection, which can miss novel or sophisticated attacks. Manual analysis of network traffic is time-consuming and prone to human error, especially with the increasing volume of data in modern networks.

Solution

I developed a tool that leverages machine learning to automatically analyze network traffic patterns and identify anomalies that may indicate security threats. The tool processes network capture files (PCAP), extracts relevant features, and applies machine learning algorithms to detect unusual patterns.

Key Features

  • PCAP file parsing and feature extraction
  • Statistical analysis of network flows
  • Machine learning-based anomaly detection
  • Classification of traffic into benign and potentially malicious
  • Visualization of traffic patterns and detected anomalies
  • Detailed reporting of suspicious activities

Technologies Used

  • Python: Core programming language
  • Scapy: For packet manipulation and analysis
  • Scikit-learn: For implementing machine learning algorithms
  • Pandas & NumPy: For data manipulation and analysis
  • Matplotlib & Seaborn: For data visualization
  • PyShark: Python wrapper for Wireshark's tshark

Implementation Details

The tool follows a modular architecture with the following components:

1. Data Collection & Preprocessing

The tool can ingest PCAP files from various sources including live captures or stored files. It preprocesses the data by:

  • Extracting packet headers and payloads
  • Organizing packets into flows based on IP addresses and ports
  • Computing statistical features for each flow
  • Normalizing features for machine learning algorithms

2. Feature Engineering

The tool extracts various features from network traffic, including:

  • Flow duration, packet count, and bytes transferred
  • Inter-arrival time statistics
  • Packet size distribution
  • TCP flags and connection patterns
  • Protocol-specific attributes

3. Machine Learning Models

Several machine learning algorithms are implemented:

  • Isolation Forest for anomaly detection
  • Random Forest for classification
  • K-means clustering for traffic pattern identification
  • DBSCAN for density-based anomaly detection

4. Visualization & Reporting

The tool provides various visualizations to help analysts understand the results:

  • Traffic flow graphs
  • Anomaly score distributions
  • Feature importance plots
  • Interactive network maps

Example Code Snippet

import pandas as pd
import numpy as np
from scapy.all import rdpcap
from sklearn.ensemble import IsolationForest

def extract_features(pcap_file):
    """Extract features from PCAP file for anomaly detection"""
    packets = rdpcap(pcap_file)
    flows = {}
    
    # Group packets into flows
    for packet in packets:
        if 'IP' in packet and ('TCP' in packet or 'UDP' in packet):
            src_ip = packet['IP'].src
            dst_ip = packet['IP'].dst
            
            if 'TCP' in packet:
                src_port = packet['TCP'].sport
                dst_port = packet['TCP'].dport
                proto = 'TCP'
            else:
                src_port = packet['UDP'].sport
                dst_port = packet['UDP'].dport
                proto = 'UDP'
                
            flow_id = f"{src_ip}:{src_port}-{dst_ip}:{dst_port}-{proto}"
            
            if flow_id not in flows:
                flows[flow_id] = []
                
            flows[flow_id].append({
                'time': packet.time,
                'size': len(packet),
                'flags': packet.flags if 'TCP' in packet else None
            })
    
    # Calculate features for each flow
    flow_features = []
    for flow_id, packets in flows.items():
        if len(packets) < 2:
            continue
            
        # Sort packets by time
        packets.sort(key=lambda x: x['time'])
        
        # Calculate features
        duration = packets[-1]['time'] - packets[0]['time']
        packet_count = len(packets)
        bytes_transferred = sum(p['size'] for p in packets)
        
        # Calculate inter-arrival times
        inter_arrival_times = [packets[i+1]['time'] - packets[i]['time'] 
                              for i in range(len(packets)-1)]
        
        # Add features to list
        flow_features.append({
            'flow_id': flow_id,
            'duration': duration,
            'packet_count': packet_count,
            'bytes_transferred': bytes_transferred,
            'mean_iat': np.mean(inter_arrival_times) if inter_arrival_times else 0,
            'std_iat': np.std(inter_arrival_times) if inter_arrival_times else 0,
            'mean_size': np.mean([p['size'] for p in packets]),
            'std_size': np.std([p['size'] for p in packets])
        })
    
    return pd.DataFrame(flow_features)

def detect_anomalies(features, contamination=0.05):
    """Detect anomalies in network flows using Isolation Forest"""
    # Select numerical features
    X = features.drop('flow_id', axis=1)
    
    # Train isolation forest
    model = IsolationForest(contamination=contamination, random_state=42)
    features['anomaly_score'] = model.fit_predict(X)
    
    # Convert predictions: -1 for anomalies, 1 for normal
    features['is_anomaly'] = features['anomaly_score'].apply(lambda x: x == -1)
    
    return features

# Example usage
if __name__ == "__main__":
    pcap_file = "network_capture.pcap"
    features = extract_features(pcap_file)
    results = detect_anomalies(features)
    
    # Print anomalies
    anomalies = results[results['is_anomaly']]
    print(f"Detected {len(anomalies)} anomalous flows out of {len(results)}")
    print(anomalies[['flow_id', 'duration', 'packet_count', 'bytes_transferred']])
